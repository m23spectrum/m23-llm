# Security Policy

## Supported Versions

Only the latest commit on the `main` branch is actively supported and maintained.

| Version | Supported          |
| ------- | ------------------ |
| latest  | :white_check_mark: |

## Reporting a Vulnerability

If you find any security vulnerabilities in the mathematical routines, dataset streaming implementations, or model integrations, please do not file a public Issue. Instead, contact the maintainers directly or open a draft security advisory via GitHub if possible.
